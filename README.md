This add-on provides a block that features a big image on top and a title and description editable inline.

### Prerequisites

- Docker
- Node 18 (e.g. via nvm)

### Build a project

Run

````
make start-dev
````

### Stop Backend (Docker)

Run:

````
make stop-backend-docker
````
